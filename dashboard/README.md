# ⚕️ Klinik Rehber & İlaç Etkileşim Güvenliği RAG Asistanı - Ön Yüz (Frontend)

Bu dizin, projenin **Sprint 1** aşaması için geliştirilen Streamlit tabanlı kullanıcı arayüzünü (UI) içermektedir. Arayüz profesyonel ve kurumsal bir **Klinik Karar Destek Paneli** olarak tasarlanmıştır.

## 🚀 Yerelde Çalıştırma Adımları

Uygulamayı kendi bilgisayarınızda test etmek için aşağıdaki adımları sırasıyla takip edebilirsiniz:

### 1. Gerekli Kütüphanelerin Yüklenmesi

Öncelikle terminalinizde Streamlit kütüphanesinin yüklü olduğundan emin olun:
pip install streamlit

### 2. Uygulamanın Başlatılması

Proje dizinindeyken terminale aşağıdaki komutu yazarak localhost üzerinde uygulamayı ayağa kaldırabilirsiniz:

streamlit run app.py
Uygulama başlatıldığında tarayıcınızda otomatik olarak http://localhost:8501 adresi açılacaktır.
